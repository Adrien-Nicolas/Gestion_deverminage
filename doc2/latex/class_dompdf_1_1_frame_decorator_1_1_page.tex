{\footnotesize\ttfamily \+\_\+page\+\_\+break\+\_\+allowed (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{class_dompdf_1_1_frame}{Frame}}}]{\$frame }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [protected]}}

Determine if a page break is allowed before \$frame \href{http://www.w3.org/TR/CSS21/page.html\#allowed-page-breaks}{\texttt{ http\+://www.\+w3.\+org/\+TR/\+CSS21/page.\+html\#allowed-\/page-\/breaks}}

In the normal flow, page breaks can occur at the following places\+:


\begin{DoxyEnumerate}
\item In the vertical margin between block boxes. When a page break occurs here, the used values of the relevant \textquotesingle{}margin-\/top\textquotesingle{} and \textquotesingle{}margin-\/bottom\textquotesingle{} properties are set to \textquotesingle{}0\textquotesingle{}.
\item Between line boxes inside a block box.
\item Between the content edge of a block container box and the outer edges of its child content (margin edges of block-\/level children or line box edges for inline-\/level children) if there is a (non-\/zero) gap between them.
\end{DoxyEnumerate}

These breaks are subject to the following rules\+:


\begin{DoxyItemize}
\item Rule A\+: Breaking at (1) is allowed only if the \textquotesingle{}page-\/break-\/after\textquotesingle{} and \textquotesingle{}page-\/break-\/before\textquotesingle{} properties of all the elements generating boxes that meet at this margin allow it, which is when at least one of them has the value \textquotesingle{}always\textquotesingle{}, \textquotesingle{}left\textquotesingle{}, or \textquotesingle{}right\textquotesingle{}, or when all of them are \textquotesingle{}auto\textquotesingle{}.
\item Rule B\+: However, if all of them are \textquotesingle{}auto\textquotesingle{} and the nearest common ancestor of all the elements has a \textquotesingle{}page-\/break-\/inside\textquotesingle{} value of \textquotesingle{}avoid\textquotesingle{}, then breaking here is not allowed.
\item Rule C\+: Breaking at (2) is allowed only if the number of line boxes between the break and the start of the enclosing block box is the value of \textquotesingle{}orphans\textquotesingle{} or more, and the number of line boxes between the break and the end of the box is the value of \textquotesingle{}widows\textquotesingle{} or more.
\item Rule D\+: In addition, breaking at (2) is allowed only if the \textquotesingle{}page-\/break-\/inside\textquotesingle{} property is \textquotesingle{}auto\textquotesingle{}.
\end{DoxyItemize}

If the above doesn\textquotesingle{}t provide enough break points to keep content from overflowing the page boxes, then rules B and D are dropped in order to find additional breakpoints.

If that still does not lead to sufficient break points, rules A and C are dropped as well, to find still more break points.

We will also allow breaks between table rows. However, when splitting a table, the table headers should carry over to the next page (but they don\textquotesingle{}t yet).
